# 明日旋律「Arknights Rhythm」

## 介绍 Introduction

仿《明日方舟》UI设计的同人音游，玩法采取经典下落式玩法，无固定轨道（Deemo2类似），初定11首曲目+1首隐藏曲，每个曲目设有3-4个不同难度（隐藏曲1个难度），游戏计划在移动端发布（安卓、IOS）
